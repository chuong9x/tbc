F# Programming Contest

Kean Walmsley just announced a programming contest using the exciting new functional .NET language F#. Since functional programming has many uses in all realms including architecture, there should be lots of scope for making use of it in conjunction with the Revit API as well. I hope to install Visual Studio 2008 real soon now and have a look at setting up a basic F# Revit add-in project.
